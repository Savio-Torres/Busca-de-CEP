# Busca de CEP

Projeto desenvolvido com HTML, CSS e JavaScript, utilizando a API ViaCEP para buscar informações de endereço a partir de um CEP informado pelo usuário.

## Funcionalidades

- Digitação de CEP com máscara automática
- Busca de endereço pela API ViaCEP
- Exibição de rua, bairro, cidade, estado e DDD
- Validação de CEP
- Mensagens de erro e sucesso
- Interface responsiva

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- API ViaCEP

## Como executar

1. Clone este repositório
2. Abra o arquivo `index.html` no navegador
3. Digite um CEP válido
4. Clique em "Buscar endereço"
